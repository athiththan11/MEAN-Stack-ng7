Click on the Create new Dev button to create a new dev on the application.

The above table presents all the devs stored in our presented mLab (MongoDb). The edit button routes to an edit form to edit the details of devs.

The delete button deletes a dev permenantly from the database. The ObjectID of the dev is used to query and delete from the database.
